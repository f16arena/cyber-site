F16 HUB — Карты CS2 (Map images)
==================================

Сюда положите JPG-превью карт. Имя файла = id карты (lowercase):
  mirage.jpg
  inferno.jpg
  nuke.jpg
  ancient.jpg
  anubis.jpg
  vertigo.jpg
  dust2.jpg

Рекомендуемый размер: 640x360 (16:9), JPG quality 80.

Откуда взять:
- Screenshots Steam Workshop / CS2 official: https://store.steampowered.com/app/730/CounterStrike_2/
- HLTV map page (вверху страницы): https://www.hltv.org/maps
- Cloudflare CDN Valve (без гарантий стабильности URL):
    https://blog.counter-strike.net/wp-content/uploads/2023/03/de_mirage.jpg
    и аналогичные

Если файла нет — MapCard всё равно рендерится: показывает
цветной CSS-градиент темы карты + название. Так что добавлять
картинки опционально.
